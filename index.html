About TypeTrek

TypeTrek was inspired by the need to improve typing skills in a fun and interactive way. The project started as a personal challenge to create a comprehensive typing tutor that not only helps users type faster but also improves their accuracy. It evolved into a full-fledged application with various features aimed at making typing practice engaging and effective.

This project is a Portfolio Project for Holberton School. Feel free to check out the repository and connect with the team members.
